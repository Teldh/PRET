Visualization Description

1. Matrix:
asymmetric adjacency matrix where each cell represents a prerequisite relation between two concepts. Cells are coloured in order to visually distinguish strong and weak relations. The matrix arrangement is dynamic, i.e. the concepts along the axes can be sorted according to different criteria: order of first appearance in the text, alphabetical order, frequency and agreement rate (i.e., number of annotators that created that pair).

2. Arc Diagram:
a special kind of network graph constituted by nodes that represent concepts and links that show relationships between concepts. In arc diagrams, nodes are displayed along a single axis and links are represented with arcs.

3. Simple Graph:
network-like representation of the selected dataset: concepts are nodes in the graph while edges represent prerequisite relations.
